自定义http变量

使用 DataTables 的服务器端处理选项时，向服务器发送额外信息通常很有用。这是通过使用 ajax.data 选项来完成的，可以使用两种不同的方式之一：

  • object - 要发送到服务器的对象数据。 这对于向请求中添加静态数据非常有用
  • function - 通过函数返回要发送到服务器的对象数据，参数为发送的数据，可以通过参数添加附加数据

本示例演示 ajax 的 ajax.data 作为函数将自定义的数据发送到服务器端。

	    "data": function (d) {
	        d.myKey = "myValue";
	        // d.custom = $('#myInput').val();

	    }
	}
}));
